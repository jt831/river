# Agent Instructions

## 项目概况
- 项目名：`River`
- 引擎：`Godot 4.6`
- 渲染：`Forward Plus`
- 物理：`Jolt Physics`
- 主场景：`res://scenes/game.tscn`

## 代码结构
- `scenes/`：场景文件，当前包括 `game.tscn`、`water.tscn`、`body.tscn`、`character_body_3d.tscn`
- `scripts/`：GDScript 脚本，当前包括抓取、玩家控制、可抓取物、水域等逻辑
- `assets/`：项目资源

## 输入映射
- `jump`
- `left`
- `right`
- `up`
- `down`
- `grab`

## 工作要求
- 先阅读现有代码和场景结构，再做修改。
- 修改时尽量保持改动范围最小，优先沿用项目现有写法。
- 每次更改代码后，都要提交到 GitHub。
- 提交前先确认改动范围，避免把无关文件一起提交。
